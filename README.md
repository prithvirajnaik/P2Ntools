# **📋 P2Ntools - Clipboard Saver**  

A **lightweight clipboard monitoring app** built with **Python** and **CustomTkinter** that saves copied text to a file and provides a user-friendly GUI for managing saved logs.  

## **✨ Features**  
✔ **Clipboard Monitoring** – Automatically saves copied text to a selected file.  
✔ **File Management** – Create, delete, and edit text files within the app.  
✔ **Hotkey Support:**  
   - 🔄 **Toggle Clipboard Monitoring** → `Shift + Alt + C`  
   - 📄 **Create a New File** → `Shift + Alt + N`  
   - ❌ **Exit Program** → `Shift + Alt + Q`  
✔ **Notifications** – Receive desktop notifications when text is saved.

## **🚀 Installation**
### **1. Clone the Repository**
```sh
git clone https://github.com/prithvirajnaik/P2Ntools.git
cd P2Ntools
 ```
### **2. Install Dependencies**
```sh
pip install -r requirements.txt
 ```
### **3. Run Script**
```sh
python main.py
```
